Usage Guide

📂 모델 불러오기 드래그 앤 드롭 ▼
  1. 파일을 AssetBox 창으로 드래그 앤 드롭
  2. 마우스: 회전(좌클릭), 줌(스크롤), 패닝(우클릭)
Tip: F 키로 모델 포커스 복귀. 한글 경로/공백 지원.
🧊 Solid 뷰 1 외관 확인 ▼

전체 외관, 내장 텍스처, 실루엣을 확인합니다.

  • 메시에 구멍이나 찢어진 부분 없는지
  • 실루엣과 스케일이 자연스러운지
배경색 전환: 하단 색상 원 클릭. 어두운 배경=밝은 에셋, 밝은 배경=실루엣 확인
🔲 Wireframe 뷰 2 토폴로지 검사 ▼
  • 엣지 플로우: 관절/얼굴 주변 루프가 자연스러운지
  • 과밀 영역: 폴리곤이 과도하게 몰려있지 않은지
  • 내부 면: 숨겨진 불필요한 면이 없는지
Tip: Retopo 뷰에서 과밀 표시된 영역을 여기서 다시 확인하면 원인 파악이 쉽습니다.
🔵 Normals 뷰 3 뒤집힌 노말 감지 ▼
파란 점 (작은): 정상 노말    빨간 점 (큰): 뒤집힌 노말

뒤집힌 노말 발견 시 하단 "Flipped Normals" 버튼 → 클릭하면 해당 위치로 카메라 이동

수정: Maya: Reverse Normals / Blender: Normals > Flip / 3ds Max: Edit Normals > Flip
🌈 Normal Map 뷰 4 노말 방향 색상화 ▼
X축 Y축 Z축
  • 같은 평면에서 색상 급변 = 스무딩 문제
  • 하드 엣지 경계의 색상 구분은 정상
🏁 UV 뷰 5 텍셀 밀도 확인 ▼
  • 정사각형 균일: 양호한 UV
  • 늘어남/찌그러짐: UV 스트레칭/왜곡
  • 크기 차이: 텍셀 밀도 불균일
Tip: 중요 부위(얼굴, 무기)의 체커가 더 작아야 합니다. 시임은 안 보이는 곳에 배치.

💾 리포트 & 썸네일 내보내기 ▼

Report: HTML 검증 보고서 → 모델러에게 수정사항 전달, 품질 이력 관리

🔄 실무 워크플로우 QA / 중간점검 / 스캔평가 ▼

에셋 납품 검수

  1. 드롭 → 종합 판정 → 빨간/노란 항목 확인
  2. Normals(3) → UV(5) → Retopo(6)
  3. Report 생성 → 모델러에게 전달

모델링 중간 점검

  1. DCC 익스포트 → AssetBox 드롭
  2. Wire(2) → Normals(3) → UV(5) → Retopo(6)

스캔 데이터 평가

  1. 스캔 메시 드롭 → Retopo(6) 진단
  2. 리토폴로지 도구 처리 후 재확인
📂 Loading Models Drag & Drop ▼
  1. Drag and drop a file onto the window
  2. Mouse: rotate (left), zoom (scroll), pan (right)
Tip: Press F to refocus on model. Unicode paths supported.
🧊 Solid View 1 Appearance check ▼
  • Check for holes, tears, and silhouette quality
  • Verify embedded textures and consistent scale
Background: Click color circles. Dark=light assets, Light=silhouette check
🔲 Wireframe View 2 Topology inspection ▼
  • Edge flow: Natural loops around joints and faces
  • Density: No excessive concentration
  • Internal faces: No hidden geometry
Tip: Cross-reference Retopo view's over-dense areas here for root cause.
🔵 Normals View 3 Flipped normal detection ▼
Blue (small): Correct    Red (large): Flipped

"Flipped Normals" button → click to fly camera to problem area

Fix: Maya: Reverse Normals / Blender: Normals > Flip / 3ds Max: Edit Normals > Flip
🌈 Normal Map View 4 RGB normal direction ▼
X-axis Y-axis Z-axis
  • Abrupt color changes on flat surfaces = smoothing issues
  • Distinct separation at hard edges is normal
🏁 UV View 5 Texel density check ▼
  • Uniform squares: Good UVs
  • Stretched/distorted: UV issues
  • Size variation: Uneven texel density
Tip: Important areas should have smaller checkers. Place seams in hidden areas.

🔄 Workflow Examples QA / Check / Scan ▼

Asset Delivery QA

  1. Drop → verdict → review red/yellow items
  2. Normals(3) → UV(5) → Retopo(6)
  3. Generate Report → send to modeler

Mid-Production Check

  1. DCC export → drop into AssetBox
  2. Wire(2) → Normals(3) → UV(5) → Retopo(6)

Scan Data Evaluation

  1. Drop scan → Retopo(6) diagnosis
  2. Retopo tool → re-check in AssetBox

Installation Guide

AssetBox is an open-source project and is not code-signed. Your OS may show a security warning on first launch.

  1. Download the .dmg file and open it
  2. Drag AssetBox to the Applications folder
  3. Open Terminal and run:
xattr -cr /Applications/AssetBox.app
  1. Now open AssetBox normally from Applications
Why? macOS blocks unsigned apps with "damaged" error. The command above removes the quarantine flag. This is safe for open-source apps you trust.
  1. Download the .msi file and run it
  2. "Windows protected your PC" (SmartScreen) will appear
  3. Click "More info"
  4. Click "Run anyway"
  5. Follow the installer. This warning only appears once.
Alternative: Download the .exe file for a portable version (no install needed).
